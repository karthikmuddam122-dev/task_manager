# 🧑‍💻 Team Task Manager

A simple full-stack web application for managing team projects and tasks. Users can register, log in, create projects, assign tasks, and track progress using a dashboard.

---

## 🚀 Features

- User Signup & Login
- Role-based structure (Admin/Member support ready)
- Create Projects
- Create and Manage Tasks
- Task Status Update (To Do / Done)
- Dashboard with task statistics
- SQLite database integration
- REST API backend using Flask

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: SQLite
- API: RESTful APIs
